🔗 شرح Attribute Binding في جافا سكريبت

مقدمة عن Attributes (الخصائص) 📚

ما هي ال Attributes؟ 🤔

الـ Attributes هي خصائص إضافية نضيفها لعناصر HTML لتحديد سلوكها ومظهرها. مثلاً: عندما نضيف صورة، نحتاج لتحديد مصدر الصورة باستخدام خاصية src.

🗺️ خريطة تعلم الـ Attributes:

4
خصائص النصوص التوضيحية placeholder و alt

تستخدم هذه الخصائص لتحسين تجربة المستخدم وإمكانية الوصول:

  • placeholder: نص توضيحي يظهر في حقول الإدخال
  • alt: نص بديل للصور عندما لا يمكن عرضها
<input placeholder="أدخل اسمك"> <img alt="صورة توضيحية">
3
خصائص العناصر التفاعلية disabled و value و type

تتحكم هذه الخصائص في سلوك العناصر التفاعلية:

  • disabled: تعطيل العنصر
  • value: القيمة الحالية للعنصر
  • type: نوع العنصر (مثل text, number, email)
<button disabled>زر معطل</button> <input type="email" value="example@email.com">
2
خصائص الروابط والمصادر src و href

تستخدم لتحديد مصادر الوسائط والروابط:

  • src: مصدر الصور والفيديو والصوت
  • href: وجهة الروابط
<img src="image.jpg"> <a href="https://example.com">رابط</a>
1
خصائص التحكم في العناصر id و class

تستخدم لتحديد وتنسيق العناصر:

  • id: معرف فريد للعنصر
  • class: فئة أو مجموعة من الأنماط
<div id="main"></div> <div class="container"></div>

📝 شرح المفاهيم الأساسية

  • Attribute Binding:

    هو عملية برمجية تمكننا من ربط خصائص عناصر HTML (مثل src, href, disabled, id, class) مع البيانات في JavaScript بشكل تفاعلي.
    بمعنى آخر: عندما تتغير البيانات في JavaScript، تتغير خصائص العناصر تلقائياً معها.

    🎯 الأهداف الرئيسية:

    • واجهة مستخدم تفاعلية: تحديث العناصر تلقائياً دون تدخل يدوي
    • استجابة لتفاعلات المستخدم: تغيير تصميم أو محتوى العناصر بناءً على نشاط المستخدم
    • تزامن مع حالة التطبيق: إدارة سهلة لحالة التطبيق وواجهته

    ⚙️ كيف يعمل Attribute Binding؟

    1. تحديد العنصر المراد تغيير خصائصه باستخدام JavaScript
    2. ربط متغير في الحالة الحالية (state) بخاصية العنصر
    3. عند تغيير قيمة المتغير، يتم تحديث العنصر تلقائياً

    💡 الخلاصة:

    تجعل هذه العملية العمل مع واجهات المستخدم أسرع وأكثر مرونة وتفاعلاً، مما يعزز تجربة المستخدم ويقلل الجهد البرمجي.

🎯 أمثلة تفاعلية على Attribute Binding

4
تحديث النصوص value & placeholder
مثال: تحديث قيمة حقل الإدخال
// HTML
<input type="text" id="inputField" 
       placeholder="أدخل نصاً هنا">

// JavaScript
function updatePlaceholder() {
    const input = document.getElementById('inputField');
    input.placeholder = placeholders[currentIndex];
}


3
تغيير الأنماط class attribute
مثال: تغيير نمط العنصر
// HTML
<div id="styleDemo" class="demo-box">مرحباً!</div>

// JavaScript
function changeStyle(style) {
    const demo = document.getElementById('styleDemo');
    demo.className = 'demo-box ' + style;
}
مرحباً!

2
تفعيل/تعطيل الأزرار disabled attribute
مثال: تفعيل وتعطيل الزر
// HTML
<button id="targetBtn" disabled="false">
    زر تجريبي
</button>

// JavaScript
function toggleButton() {
    const btn = document.getElementById('targetBtn');
    btn.disabled = !btn.disabled;
}


1
تغيير الصور src attribute
مثال: تغيير الصورة عند النقر
// HTML
<img id="demoImage1" 
     src="image1.jpg" 
     alt="صورة توضيحية">

// JavaScript
function changeImage() {
    const img = document.getElementById('demoImage1');
    img.src = images[currentImageIndex];
}
