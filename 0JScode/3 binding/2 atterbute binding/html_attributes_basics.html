Understanding HTML Attributes

What is an HTML Attribute?

An attribute is extra information we give to HTML elements. Think of attributes as special properties that tell elements how to look or behave.

Basic Structure:

<element attribute="value">content</element>

Common HTML Attributes Examples

1. class attribute

This text is red because of its class attribute

This text is blue because of its class attribute

<p class="red-text">This text is red</p>

2. id attribute

Every id must be unique in the page

<p id="unique-paragraph">Content</p>

3. src attribute


<img src="image-path.jpg" alt="Description">

4. href attribute

Click me
<a href="https://www.example.com">Click me</a>

5. disabled attribute


<button disabled>Can't click me</button>

Common HTML Attributes Table

Attribute Used On Purpose
class Any element Add styles or identify elements
id Any element Uniquely identify an element
src img, script Specify source file/URL
href a, link Specify link destination
disabled input, button Disable user interaction
style Any element Add inline CSS styles

Key Points About Attributes:

  • Attributes always go in the opening tag
  • Most attributes need values (like class="value")
  • Some attributes don't need values (like disabled)
  • Attribute values should be in quotes
  • One element can have multiple attributes

Example with Multiple Attributes:


<button id="special-btn" class="big-button" disabled>Multiple Attributes</button>
