2. Simple React-Style Example

// React example (for demonstration)
function Button() {
    const [isEnabled, setIsEnabled] = useState(true);

    return (
        <button 
            className={`button ${isEnabled ? 'enabled' : 'disabled'}`}
            disabled={!isEnabled}
            onClick={() => setIsEnabled(!isEnabled)}
        >
            {isEnabled ? 'Click Me' : 'Disabled'}
        </button>
    );
}

What is Attribute Binding?

  • Attribute binding means connecting HTML attributes to your JavaScript variables
  • When the variable changes, the attribute automatically updates
  • Common attributes to bind:
    • class - for styling
    • disabled - for form controls
    • src - for images
    • href - for links
    • value - for input fields

Key Differences Between Frameworks:

Framework How to Bind Example
Vanilla JS Direct DOM manipulation element.setAttribute('class', value)
React JSX attributes className={value}
Vue v-bind directive :class="value"
