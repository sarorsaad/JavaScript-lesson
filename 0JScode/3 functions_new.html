JavaScript Functions

Function Definition

A function definition in JavaScript describes the structure of a function, specifying what it does when invoked. Functions allow us to encapsulate reusable blocks of code.

Syntax of Function Definition

function functionName(parameters) {
    // Function body: the code to execute
}

Components

  • function keyword: Required for defining a function.
  • Function name: Used to call (invoke) the function.
  • Parameters: Optional placeholders that accept values when the function is called.
  • Function body: The block of code that specifies what the function does.
  • Return statement: (Optional) Used to return a value from the function.

Example of a Function Definition

function greetUser(name) {
    console.log("Hello, " + name + "!");
}

Types of Functions

1. Named Functions

function greet() {
    console.log("Hello!");
}

2. Anonymous Functions

let sayHi = function() {
    console.log("Hi!");
};

3. Arrow Functions

const multiply = (x, y) => x * y;

Predefined Functions in JavaScript

JavaScript provides many built-in functions that you can use directly without defining them first. These functions are part of the JavaScript standard library and are available across all JavaScript environments.

Note: Predefined functions are essential tools that help you perform common programming tasks without writing the functionality from scratch.

Common Predefined Functions

  • alert() - Displays an alert dialog box
  • console.log() - Outputs messages to the console
  • Math.pow() - Calculates exponentiation
  • parseInt() and parseFloat() - Convert strings to numbers
  • document.querySelector() - Selects DOM elements

Math Functions Examples

// Square root
let squareRoot = Math.sqrt(16); // Returns 4

// Power
let power = Math.pow(2, 3);    // Returns 8

// Random number
let random = Math.random();     // Returns number between 0 and 1

// Round numbers
let rounded = Math.round(3.7);  // Returns 4
