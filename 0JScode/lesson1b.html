Understanding the DOM (Document Object Model)

What is the DOM?

The DOM (Document Object Model) is a programming interface for HTML documents. It represents the page as a tree-like structure where each node is an object representing a part of the document.

DOM Tree Structure

  • document - The root of the DOM tree
  • <html> - The root element
    • <head> - Contains metadata
      • <title> - Page title
    • <body> - Contains visible content
      • <h1> - Headings
      • <a> - Links with href attribute
