Understanding Variables

Variables are containers for storing data values. JavaScript provides three ways to declare variables:

// Using var (function-scoped)
var name = "John";

// Using let (block-scoped)
let age = 25;

// Using const (constant value)
const PI = 3.14159;

Primitive Data Types

// String
let text = "Hello World";

// Number
let count = 42;
let price = 19.99;

// Boolean
let isActive = true;

// Undefined
let undefinedVar;

// Null
let emptyValue = null;

// Symbol
let uniqueId = Symbol('id');

// BigInt
let bigNumber = 9007199254740991n;
Pro Tip: Always use const by default, and only use let when you need to reassign values. Avoid using var in modern JavaScript.
