Understanding Data Structures & Control Flow

Image 1: Data Storage

Variables

Store single values of different types

let name = "John";
let age = 25;
let isActive = true;

Arrays

Store ordered collections of values

let colors = ['red', 'green', 'blue'];
let numbers = [1, 2, 3, 4, 5];

Objects

Store structured data with key-value pairs

let user = {
  name: "John",
  age: 25
};

Image 2: Control Flow

Functions

Process and transform data

function greet(name) {
  return `Hello, ${name}!`;
}

Conditions

Make decisions based on data

if (age >= 18) {
  return "Adult";
} else {
  return "Minor";
}

Loops

Process collections of data

for (let num of numbers) {
  console.log(num * 2);
}

How They Work Together

• Functions operate on data stored in Variables, Arrays, and Objects

• Conditions make decisions based on stored data values

• Loops iterate over Arrays and Object properties
