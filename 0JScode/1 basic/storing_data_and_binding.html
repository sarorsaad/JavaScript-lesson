JavaScript Data Storage and Binding

1. Storing Data in Memory

In JavaScript, we can store data in memory using variables, objects, and arrays.

Example 1: Simple Data Storage

// Storing simple data types in memory let name = "John Doe"; let age = 30; let isActive = true;

Example 2: Complex Data Storage (Objects & Arrays)

// Storing complex data structures let user = { name: "Jane Smith", age: 25, hobbies: ["reading", "coding", "hiking"], address: { street: "123 Main St", city: "Tech City" } };

2. Data Binding

Data binding connects data in memory with UI elements for automatic synchronization.

Example 3: One-way Binding

// One-way binding example let userData = { name: "Guest" }; function updateName() { const newName = document.getElementById('nameInput').value; userData.name = newName; // Update UI document.getElementById('nameDisplay').textContent = userData.name; }
Name will appear here

Example 4: Two-way Binding

// Two-way binding example const twoWayInput = document.getElementById('twoWayInput'); const twoWayOutput = document.getElementById('twoWayOutput'); twoWayInput.addEventListener('input', function(e) { twoWayOutput.textContent = e.target.value; });
Text will mirror here
