Step 5: Complete DOM Example

Let's see how to combine element selection, modification, and events together in one example. We'll create all content using JavaScript!

HTML Code:

<!-- Just an empty div with an ID -->
<div id="root"></div>

JavaScript Code:

// 1. Select the element
const rootElement = document.getElementById('root');

// 2. Create and append new elements
const title = document.createElement('h3');
title.textContent = 'Interactive Element';
rootElement.appendChild(title);

const paragraph = document.createElement('p');
paragraph.textContent = 'Click me or hover over me!';
rootElement.appendChild(paragraph);

// 3. Modify the element's style
rootElement.style.padding = '20px';
rootElement.style.border = '2px solid #4CAF50';
rootElement.style.borderRadius = '8px';
rootElement.style.cursor = 'pointer';
rootElement.style.backgroundColor = '#1e1e1e';
rootElement.style.transition = 'all 0.3s ease';
rootElement.style.fontFamily = 'Arial, sans-serif';
rootElement.style.color = '#ffffff';
rootElement.style.fontSize = '18px';
rootElement.style.margin = '20px';
rootElement.style.maxWidth = '600px';

// Style the title and paragraph
title.style.fontSize = '24px';
title.style.color = '#4CAF50';
title.style.marginBottom = '10px';

paragraph.style.fontSize = '18px';
paragraph.style.color = '#ddd';
paragraph.style.lineHeight = '1.5';

// 4. Add multiple event listeners
rootElement.addEventListener('click', function() {
    this.style.backgroundColor = getRandomColor();
    const timeStamp = document.createElement('p');
    timeStamp.textContent = 'Clicked at: ' + new Date().toLocaleTimeString();
    timeStamp.style.color = '#4CAF50';
    timeStamp.style.fontSize = '16px';
    timeStamp.style.marginTop = '5px';
    this.appendChild(timeStamp);
});

rootElement.addEventListener('mouseover', function() {
    this.style.transform = 'scale(1.05)';
});

rootElement.addEventListener('mouseout', function() {
    this.style.transform = 'scale(1)';
});

// Helper function for random colors
function getRandomColor() {
    const letters = '0123456789ABCDEF';
    let color = '#';
    for (let i = 0; i < 6; i++) {
        color += letters[Math.floor(Math.random() * 16)];
    }
    return color;
}

What's Happening Here?

  1. We start with an empty div that has id="root"
  2. Using JavaScript, we:
    • Create new elements (createElement)
    • Set their content (textContent)
    • Append them to our root element (appendChild)
  3. We style the element entirely through JavaScript
  4. We add three different event listeners:
    • click: Changes background color and adds a timestamp
    • mouseover: Makes the element slightly larger
    • mouseout: Returns the element to its original size

Step 6: Converting to React

Why Convert to React?

Converting from pure JavaScript to React offers several significant advantages:

  • Component-Based Architecture: React's component structure makes code more reusable, maintainable, and easier to test.
  • Virtual DOM: React's virtual DOM optimizes rendering performance by minimizing actual DOM manipulations.
  • State Management: React provides efficient ways to manage application state with hooks and context.
  • Declarative Syntax: Write what you want to achieve rather than how to achieve it, making code more readable.
  • Rich Ecosystem: Access to a vast collection of libraries, tools, and community support.
