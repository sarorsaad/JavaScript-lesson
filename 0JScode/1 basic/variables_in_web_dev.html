JavaScript Variable Syntax Patterns

Type Keyword Name Rules Assignment Value & Special Symbols Example
Regular Variables let
const
var 		- Start with letter, $ or _
- Can contain numbers
- Case sensitive 		= - Strings: " " or ' '
- Numbers: no quotes
- Boolean: true/false 		let age = 30; const name = "Alice";
Arrays let
const 		- Plural names preferred
- Descriptive of content 		= - Square brackets [ ]
- Comma-separated values
- Any data type 		let colors = ["red", "green"]; const nums = [1, 2, 3];
Objects let
const 		- Singular names preferred
- Describes object type 		= - Curly braces { }
- Key-value pairs
- Colon : separator
- Comma between pairs 		let person = { name: "Alice", age: 30 };

Important Notes:

  • The = operator is for assignment, not comparison (use == or === for comparison)
  • Use const by default, let when the value needs to change
  • Avoid using var in modern JavaScript
