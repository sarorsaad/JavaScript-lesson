Web Development Fundamentals - Lesson 1A

1. Basic JavaScript Concepts

  • Variables:
    • var - Traditional variable declaration (function-scoped)
    • let - Modern variable declaration (block-scoped)
    • const - Constant value declaration (cannot be reassigned)
  • Data Types:
    • Strings - Text values: "Hello World"
    • Numbers - Numeric values: 42, 3.14
    • Booleans - true or false
    • Arrays - Lists: [1, 2, 3]
    • Objects - Key-value pairs: {name: "John", age: 25}
  • Comments:
    • Single-line comments: // This is a comment
    • Multi-line comments: /* This is a multi-line comment */
