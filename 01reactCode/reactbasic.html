React Basic Lessons

Step 1: Create React Container

To start using React, we need to create a root container where our React components will be rendered.
// Create a root container
const root = ReactDOM . createRoot ( document . getElementById ( 'root' ));
Component Hierarchy
<App />
<Component1 />
<Component2 />
<Component3 />

Step 2: Understanding React Rendering Process

Rendering in React is the process of displaying content on the screen.
This is done by render() method.
// Render content to root container
root . render ( < App /> );

Step 3: Rendering Single React Element

// Render single element
root . render (
< h1 > Hello, World! </ h1 >
);

Step 4: Render Multiple React Elements

// Render multiple elements wrapped in a single parent container
root . render (
< div >
< h1 > Hello, World! </ h1 >
< p > This is a paragraph. </ p >

Step 5: Introduction to JSX

JSX: Write HTML-like code within JavaScript
JSX stands for JavaScript XML, and it's a syntax extension for JavaScript used in React. It allows you to write HTML-like code directly in JavaScript.

JSX Syntax

// JSX Example
root . render (
< div >
< h1 > Hello, World! </ h1 >
< p > This is JSX! </ p >
</ div >
);

Equivalent JavaScript

// Without JSX
root . render (
React.createElement ( 'div' , null, [
React.createElement ( 'h1' , null, 'Hello, World!' ),
React.createElement ( 'p' , null, 'This is JSX!' )
]);
