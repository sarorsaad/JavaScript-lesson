Step 5: Introduction to JSX

JSX: Write HTML-like code within JavaScript
JSX stands for JavaScript XML, and it's a syntax extension for JavaScript used in React. It allows you to write HTML-like code directly in JavaScript.

JSX Syntax

// JSX Example
root . render (
< div >
< h1 > Hello, World! </ h1 >
< p > This is JSX! </ p >
</ div >
);

Equivalent JavaScript

// Without JSX
root . render (
React.createElement ( 'div' , null, [
React.createElement ( 'h1' , null, 'Hello, World!' ),
React.createElement ( 'p' , null, 'This is JSX!' )
]);

Step 6: JSX Expressions

JSX Expressions

JSX Expressions represent any piece of JavaScript code that produces a value. An expression is code that resolves to a value.

Key Points:

  • Always use curly braces {} to embed JavaScript expressions in JSX
  • Variables must be declared before they can be used
  • Functions must be defined before they can be called
  • Any valid JavaScript expression can go inside the curly braces
const root = ReactDOM.createRoot(document.getElementById('root')); // These all produce values (expressions): const name = "React"; const number = 42; const isLoggedIn = true; const getGreeting = () => { return "Welcome back!"; }; root.render( <div> {/* Variable expression - resolves to "React" */} <p>Name: {name}</p> {/* Mathematical expression - resolves to 50 */} <p>Result: {number + 8}</p> {/* Function expression - resolves to "Welcome back!" */} <p>Greeting: {getGreeting()}</p> </div> );
