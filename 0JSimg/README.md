## Image Guidelines

1. **Format**:
   - Use PNG for diagrams
   - JPG for photos
   - SVG for scalable graphics

2. **Size**:
   - Max width: 1200px
   - Max height: 800px
   - Resolution: 72dpi

3. **Style**:
   - Consistent color scheme
   - Clear typography
   - High contrast

## Usage in Lessons

1. **Presentation**:
   - Start with diagram
   - Explain components
   - Show related code

2. **Documentation**:
   - Reference image names in code
   - Link to source files
   - Note any updates

3. **Maintenance**:
   - Regular format checks
   - Update outdated images
   - Version control
