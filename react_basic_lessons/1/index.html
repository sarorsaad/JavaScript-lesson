Step 4: Render Multiple React Elements (with a parent container)

React Code

// Create a root
const root = ReactDOM . createRoot ( document . getElementById ( 'root' ));
// Render multiple elements wrapped in a single parent container
root . render (
< div >
< h1 > Hello, World! </ h1 >
< p > This is a paragraph. </ p >
< button > Click Me! </ button >
</ div >
);
// Here, the elements <h1>, <p>, and <button> are returned as children of one parent container (<div>).

Step 5: Introduction to JSX

JSX: Write HTML-like code within JavaScript
JSX stands for JavaScript XML, and it's a syntax extension for JavaScript used in React. It allows you to write HTML-like code directly in JavaScript.

JSX Syntax

// JSX Example
root . render (
< div >
< h1 > Hello, World! </ h1 >
< p > This is JSX! </ p >
</ div >
);

Equivalent JavaScript

// Without JSX
root . render (
React.createElement ( 'div' , null, [
React.createElement ( 'h1' , null, 'Hello, World!' ),
React.createElement ( 'p' , null, 'This is JSX!' )
]);
