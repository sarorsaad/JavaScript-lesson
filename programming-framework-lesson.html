2. Data Types and Storage

Understanding Data Types

JavaScript has several basic data types that determine how data is stored and manipulated:

  • String: Text data
  • Number: Numeric data
  • Boolean: true/false values
  • Array: Ordered collections
  • Object: Key-value pairs
// Data Types Examples let text = "Hello World"; // String let number = 42; // Number let isActive = true; // Boolean let colors = ["red", "blue"]; // Array let user = { // Object name: "John", age: 30 };

3. Functions

Function Basics

Functions are reusable blocks of code that perform specific tasks. They can:

  • Accept input parameters
  • Process data
  • Return results
  • Contain loops and conditionals
// Function Example function processData(data) { let result = ''; // Using conditional if (typeof data === 'string') { result = data.toUpperCase(); } // Using loop for (let i = 0; i < 3; i++) { result += '!'; } return result; }

4. Arrays and Loops

Working with Arrays and Loops

Arrays store collections of data, and loops help us process each item efficiently.

// Array and Loop Example let fruits = ['apple', 'banana', 'orange']; for (let i = 0; i < fruits.length; i++) { console.log(fruits[i].toUpperCase()); }

5. Conditionals

Decision Making in Code

Conditionals allow programs to make decisions based on data values and conditions.

// Conditional Example function checkNumber(num) { if (num > 0) { return "Positive"; } else if (num < 0) { return "Negative"; } else { return "Zero"; } }

6. Putting It All Together

Complete Application Example

This example demonstrates how functions, loops, and conditionals work together in a real application.

// Task Manager Example let tasks = []; function addTask(task) { if (task.trim() !== '') { tasks.push({ id: tasks.length + 1, text: task, completed: false }); return true; } return false; } function processTasks() { let summary = { total: 0, completed: 0 }; for (let task of tasks) { summary.total++; if (task.completed) { summary.completed++; } } return summary; }
