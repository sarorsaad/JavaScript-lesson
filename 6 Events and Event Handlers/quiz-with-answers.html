DOM Manipulation Quiz - With Answers

  1. What is the most efficient way to select an element by its ID?




    Explanation: getElementById('id') is the most direct and efficient way to select an element by its ID

  2. What's the difference between querySelector() and querySelectorAll()?




    Explanation: querySelector returns only the first match, while querySelectorAll returns all matching elements

  3. How do you select all paragraphs with class 'highlight'?




    Explanation: querySelector('p.highlight') correctly selects paragraphs with class 'highlight'

  4. Why is it important to store DOM elements in variables?




    Explanation: Storing elements in variables improves performance by avoiding repeated DOM queries

  5. What type of scope does 'let' create?




    Explanation: let creates block-scoped variables

  6. Which properties can be used to change text content?




    Explanation: All three methods (innerText, innerHTML, textContent) can change text content

  7. What's the difference between innerHTML and textContent?




    Explanation: innerHTML parses HTML tags, while textContent treats them as plain text

  8. What's the modern way to add event listeners?




    Explanation: addEventListener is the modern way to add event listeners

  9. What is event bubbling?




    Explanation: Event bubbling means events propagate up through ancestor elements

  10. How do you stop event bubbling?




    Explanation: stopPropagation() prevents event bubbling

  11. How do you create a new element?




    Explanation: createElement() is the standard method to create new elements

  12. How do you add a child element to a parent?




    Explanation: appendChild() adds a child element to a parent element

  13. What's the difference between HTMLCollection and NodeList?




    Explanation: HTMLCollection is live-updating, NodeList is generally static

  14. What does preventDefault() do?




    Explanation: preventDefault() stops the default browser behavior for an event

  15. Which methods can remove an element from the DOM?




    Explanation: Both remove() and removeChild() can remove elements from the DOM

  16. What is event delegation?




    Explanation: Event delegation uses a parent element to handle events on children

  17. How do you add multiple classes at once?




    Explanation: classList.add() can add multiple classes at once

  18. What are data-* attributes used for?




    Explanation: data-* attributes are used to store custom data on HTML elements

  19. How do you check if an element has a specific class?




    Explanation: classList.contains() checks if an element has a specific class

  20. What's the difference between getAttribute and dataset?




    Explanation: getAttribute works with any attribute, while dataset is specifically for data-* attributes
