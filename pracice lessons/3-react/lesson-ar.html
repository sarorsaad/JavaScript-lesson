شرح React للمبتدئين

سنتعلم في هذا الدرس كيفية استخدام React لبناء واجهات مستخدم تفاعلية

١. الإعداد الأساسي

index.html
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>React App</title> <script src="https://unpkg.com/react@17/umd/react.development.js"></script> <script src="https://unpkg.com/react-dom@17/umd/react-dom.development.js"></script> <script src="https://unpkg.com/babel-standalone@6/babel.min.js"></script> </head> <body> <div id="root"></div> <script type="text/babel" src="App.js"></script> </body> </html>
  • إضافة مكتبات React الأساسية
  • إضافة Babel لترجمة JSX
  • إنشاء عنصر root لتركيب التطبيق
  • ربط ملف App.js

٢. المكونات (header Component)

App.js - Basic Component
const App = () => { return( <div> <h2>sarorAcademy</h2> <p>this is create website with stastic code</p> </div> ) } const root = ReactDOM.createRoot(document.getElementById("root")); root.render(<App/>);
  • تعريف مكون App باستخدام Arrow Function
  • استخدام JSX لكتابة HTML داخل JavaScript
  • إنشاء root للتطبيق باستخدام createRoot
  • تركيب المكون App في العنصر root

٣. الخصائص (Props)

App.js - Using Props
const App = () => { return ( <Header title="مشروعي الجميل" subtitle="هذا هو مشروعي الأول بـ React" /> ); };
  • تمرير البيانات للمكون عبر Props
  • استخدام علامة self-closing للمكون
  • يمكن تمرير أي نوع من البيانات

٤. التكرار والمصفوفات

App.js - Mapping Arrays
const sections = [ { title: 'القسم الأول', content: 'محتوى القسم الأول' }, { title: 'القسم الثاني', content: 'محتوى القسم الثاني' } ]; const App = () => { return ( <div> {sections.map((section, index) => ( <Section key={index} title={section.title} content={section.content} /> ))} </div> ); };
  • استخدام map() لتكرار المكونات
  • إضافة key لكل عنصر مكرر
  • تمرير بيانات كل قسم كـ props

٥. معالجة الأحداث

App.js - Event Handling
const LinkComponent = ({ title, linkText }) => { const handleClick = (e) => { e.preventDefault(); console.log('تم النقر على الرابط!'); }; return ( <div className="link-container"> <h1>{title}</h1> <a href="#" onClick={handleClick}> {linkText} </a> </div> ); };
  • تعريف دالة معالجة الحدث
  • استخدام onClick لربط الحدث
  • منع السلوك الافتراضي للرابط
  • استخدام Arrow Function للحفاظ على this
