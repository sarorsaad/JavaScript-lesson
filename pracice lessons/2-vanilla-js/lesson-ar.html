شرح JavaScript مع HTML - البرمجة الديناميكية

سنتعلم في هذا الدرس كيفية استخدام JavaScript لإنشاء صفحات ويب تفاعلية

١. الهيكل الأساسي للصفحة

١. الهيكل الأساسي للصفحة
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Vanilla JS Example</title> </head> <body> <div id="app"></div> <script src="script.js"></script> </body> </html>
  • نضع div مع id="app" ليكون نقطة البداية لتطبيقنا
  • نربط ملف JavaScript الخارجي باستخدام وسم script
  • نضع script في نهاية body لضمان تحميل كل العناصر

٢. الخطوات الأساسية لإنشاء العناصر

الخطوات الأساسية لإنشاء وإضافة العناصر:
  • إنشاء العنصر باستخدام createElement
  • تعيين الخصائص (class, id, etc)
  • إضافة المحتوى (textContent)
  • إضافة العنصر للصفحة (appendChild)
مثال على إنشاء عنصر كامل
// إنشاء العنصر const div = document.createElement('div'); // تعيين الخصائص div.className = 'container'; div.id = 'main-container'; // إضافة المحتوى div.textContent = 'مرحباً بكم'; // إضافة العنصر للصفحة document.body.appendChild(div);
  • مثال عملي يوضح كيفية إنشاء عنصر div جديد
  • إضافة class و id للعنصر
  • إضافة نص داخل العنصر
  • إضافة العنصر إلى body الصفحة

٣. إنشاء العناصر (Create Header Elements)

إنشاء عناصر HTML
const root = document.getElementById("root"); const heading = document.createElement("h1"); heading.textContent = "hello world"; root.appendChild(heading); const para = document.createElement("p"); para.textContent = "this is create website with vanilla JS"; root.appendChild(para);
  • document.getElementById: الحصول على العنصر الرئيسي من HTML
  • document.createElement: إنشاء عنصر HTML جديد
  • textContent: تعيين النص داخل العنصر
  • appendChild: إضافة العنصر الجديد إلى العنصر الرئيسي
  • يمكن إنشاء أي عنصر HTML مثل h1, p, div, span
  • يمكن تعديل خصائص العنصر مثل className, style

٤. إنشاء عناصر HTML ديناميكياً

٤. إنشاء عناصر HTML ديناميكياً
const header = document.createElement('header'); header.className = 'header'; const title = document.createElement('h1'); title.textContent = this.title; header.appendChild(title);
  • createElement: ينشئ عنصر HTML جديد
  • className: يضيف class للتنسيق
  • textContent: يضيف النص داخل العنصر
  • appendChild: يضيف عنصر داخل عنصر آخر

٥. التعامل مع المصفوفات (Arrays)

٥. التعامل مع المصفوفات (Arrays)
const sections = [ { title: 'القسم الأول', content: 'المحتوى' }, { title: 'القسم الثاني', content: 'المحتوى' } ]; for (const section of sections) { const sectionDiv = document.createElement('div'); // ... إنشاء العناصر }
  • نعرّف مصفوفة تحتوي على كائنات (objects)
  • نستخدم for...of للمرور على كل عنصر
  • ننشئ العناصر ديناميكياً لكل قسم

٦. معالجة الأحداث (Events)

٦. معالجة الأحداث (Events)
link.addEventListener('click', (e) => { e.preventDefault(); console.log('تم النقر على الرابط!'); });
  • addEventListener: يضيف مستمع للحدث
  • e.preventDefault(): يمنع السلوك الافتراضي للرابط
  • استخدام السهم => للدوال المجهولة
