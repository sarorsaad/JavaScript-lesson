شرح هيكل HTML الأساسي خطوة بخطوة

سنتعلم في هذا الدرس كيفية بناء صفحة HTML بشكل صحيح وفهم كل عنصر من عناصرها

١. إعلان نوع المستند (DOCTYPE)

DOCTYPE Declaration
<!DOCTYPE html>
  • هذا السطر يخبر المتصفح أننا نستخدم HTML5
  • ضروري في بداية كل ملف HTML

٢. عنصر HTML الرئيسي

HTML Root Element
<html lang="ar" dir="rtl"> <!-- محتوى الصفحة --> </html>
  • العنصر الجذر الذي يحتوي على كل محتوى الصفحة
  • lang="ar" لتحديد اللغة العربية
  • dir="rtl" لجعل اتجاه النص من اليمين لليسار

٣. قسم head

Head Section
<head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>عنوان الصفحة</title> <link rel="stylesheet" href="styles.css"> </head>
  • meta charset: يحدد ترميز الأحرف للصفحة
  • meta viewport: يضبط عرض الصفحة للأجهزة المحمولة
  • title: عنوان الصفحة في تبويب المتصفح
  • link: يربط ملف CSS الخارجي

٤. قسم body

Body Section
<body> <!-- محتوى الصفحة --> </body>
  • يحتوي على كل المحتوى المرئي في الصفحة

٥. الأقسام

Sections
<div class="section"> <h2>عنوان القسم</h2> <p>محتوى القسم</p> </div>
  • كل قسم يحتوي على:
  • عنوان h2 للقسم
  • فقرة p للمحتوى
  • class للتنسيق

٦. الروابط

Links
<a href="#" class="link">النص</a>
  • a: عنصر الرابط
  • href="#": وجهة الرابط (# تعني أنه لا يوجد وجهة محددة حالياً)
  • class: للتنسيق وإضافة التأثيرات مثل hover
